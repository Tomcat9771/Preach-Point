# Preach Point - Local Test Build

## Setup

1. Unzip and host on a static server (e.g. VSCode Live Server, Netlify).
2. Edit `script.js` to replace `YOUR_API_KEY_HERE` with your OpenAI key.
3. Open `index.html` in your browser.
4. Select language, passage, tone, and level; click **Generate Commentary**.
5. Use **Copy** or **Download PDF** to save results.

Enjoy your Bible study companion!
